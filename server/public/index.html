Twitter DevRel Agent API

Welcome to the Twitter DevRel Agent API documentation. This API provides endpoints for managing users, tweets, and analytics for a Twitter-like application.

API Endpoints

User Endpoints

POST /api/users/register

Register a new user

{
  "username": "johndoe",
  "email": "john@example.com",
  "password": "password123"
}

POST /api/users/login

Login a user

{
  "username": "johndoe",
  "password": "password123"
}

GET /api/users/profile

Get the current user's profile (requires authentication)

Tweet Endpoints

POST /api/tweets

Create a new tweet (requires authentication)

{
  "content": "This is my first tweet!"
}

GET /api/tweets

Get all tweets

GET /api/tweets/user/:userId

Get tweets by a specific user

Analytics Endpoints

GET /api/analytics/followers/growth/:userId

Get follower growth over time for a user

GET /api/analytics/engagement/:userId

Get tweet engagement stats for a user

GET /api/analytics/account/:userId

Get overall account stats for a user

Authentication

Most endpoints require authentication using a JWT token. Include the token in the Authorization header:

Authorization: Bearer your_jwt_token

Getting Started

To use this API:

  1. Register a new user using the /api/users/register endpoint
  2. Login with your credentials to get a JWT token
  3. Include the token in your requests to authenticated endpoints
